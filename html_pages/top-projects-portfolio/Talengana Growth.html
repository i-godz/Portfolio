Talengana Growth Analysis

Description

In this analysis project, I employed the STAR (Situation, Task, Action, Result) method to quantitatively assess Telangana's growth. By doing so, I've provided the Telangana government with valuable insights that facilitate data-informed decision-making, ultimately fueling the state's continued development and growth.

My Role

The first step was to collect the relevant data from Open Data Telangana. This dataset included data on the Registration & Stamps Department, Transportation Department, and TS-iPASS, spanning across 33 districts. Once the data was collected, it was imported into MySQL Workbench. This is a relational database management system that was used to clean and prepare the data for analysis.

The next step was to design and develop a Tableau dashboard to analyze the data. The dashboard was designed to provide the Telangana government with insights into the performance of the Registration & Stamps Department, Transportation Department, and TS-iPASS. The dashboard also included interactive filters and drill-down functionalities to allow users to explore the data in more detail.

Once the dashboard was completed, it was delivered to the Telangana government. The dashboard was hosted on Tableau Server, which allowed the government to access it from anywhere. Also a comprehensive report was produced to accompany the dashboard. The report provided a detailed overview of the findings and included recommendations for how the data could be used to improve the performance of the three departments
